1.Δημιουργια applet που θα μετατρεπει μιλια σε χλμ.
2.Αριθμηστε τις γραμμες του κωδικα και σχολιαστε την λειτουργια τους.

ΘΕΜΑ Β
Υλοποιηση προγραμματος που αποτελειται απο δυο αρχεια:μια σελιδα HTML και ενα 
java servlet. Η σελιδα HTML θα πρεπει να υλοποιει μια φορμα που θα ζητα εναν 
κωδικο και θα τον στελνει στο  servlet, το οποιο θα επικοινωνει με ενα DBMS που
περιεχει τους πινακες student(id int, name char(100), search_student(id int)).
Πρωτα θα εισαγει στον search_student μια εγγραφη με id ισο με τον κωδικο που δοθηκε
στη φορμα και μετα θα αναζητα στον πινακα student την εγγραφη του φοιτητη με αυτο το id.
Αν υπαρχει τετοιος φοιτητης η συναλλαγη θα ολοκληρωνει(commit) και θα επιστρεφεται μια HTML
σελιδα με το ονομα του φοιτητη, αλλιως η συναλλαγη θα εγκαταλειπει(abort) και θα 
επιστρεφει μια σελιδα HTML με το μηνυμα ¨δεν βρεθηκε φοιτητης¨.
